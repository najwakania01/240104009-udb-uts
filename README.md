## Konfigurasi Service
### 1. MySQL
- Container name : `uts_mysql`
- Port : `3306`
- User : `root`
- Database : `uts_db`

### 2. PostgreSQL
- Container name : `uts_postgres`
- Port : `5432`
- User : `uts_user`
- Database : `uts_pgdb`

### 3. Adminer
- Port : `8009`
- Digunakan untuk mengelola MySQL dan PostgreSQL melalui browser
